# Faker - HTTP server for testing, mocking, faking

## Handlers

* `/Mansur` - will salute you
* `/echo/EchoString` - will reply with EchoString
* `/404` - will reply with appropriate HTTP code

## Usage

```shell
% ./faker -h
Usage of ./faker:
  -code int
        response code (default 200)
  -port int
        port number (default 8080)
```
